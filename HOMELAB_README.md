# Homelab Setup

## Overview
Self-hosted infrastructure environment for learning cloud-native technologies and modern infrastructure practices.

## Infrastructure

**Hardware:**
- 3x HPE bare metal nodes (1x control plane, 2x worker nodes)
- TrueNAS server for NFS storage (PV)

**Core Technologies:**
- **Container Orchestration:** Kubernetes
- **Infrastructure as Code:** Terraform, Ansible, Helm
- **GitOps:** Flux, GitHub
- **Networking:** Flannel CNI, Cloudflare Tunnel
- **Monitoring:** Prometheus, Grafana

## Learning Goals

- Preparing for Certified Kubernetes Administrator (CKA) certification
- Building proficiency with cloud-native tools and practices
- Implementing best practices for security, observability, and automation
- Managing multi-node Kubernetes clusters
